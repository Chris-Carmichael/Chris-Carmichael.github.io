2020 Python3 and Ansible

January 11, 2020

2020 Python 3 - Leave the Past Behind

It's 2020, and Guido warned us long ago that Python 2 was being deprecated. The biggest issue is no more security updates for Python 2. Security is important, don't ya think?

Time to make some upgrades and mods.

Install Python 3

I already had this installed, but finding instructions is easy to do. This write-up from vitux.com looks pretty good.

# install the bits
sudo apt update && sudo apt install python3 -y

Python Alternatives - Default to Python 3 in Bash

You'll want to change the default behavior of your machine to use Python 3. This will be effective across all profiles.

update-alternatives --install /usr/bin/python python /usr/bin/python3 10

Update Ansible to default to Python 3

I'm an Ansible guy, and Ansible uses Python, so let's check that out. If you look at the Ansible version, you'll notice it defaulting to python-version = 2.7.17 on my machine.

ansible --version
ansible 2.9.2
  config file = /home/ansible/lab-ansible/ansible.cfg
  configured module search path = [u'/opt/modules']
  ansible python module location = /usr/lib/python2.7/dist-packages/ansible
  executable location = /usr/bin/ansible
  python version = 2.7.17 (default, Nov  7 2019, 10:07:09) [GCC 7.4.0]

Let's change that. Per Ansible Docs, we are just gonna install Ansible via pip3.

# Install ansible
sudo pip3 install ansible

# check the version
ansible --version
ansible 2.9.2
  config file = /etc/ansible/ansible.cfg
  configured module search path = ['/home/chris/.ansible/plugins/modules', '/usr/share/ansible/plugins/modules']
  ansible python module location = /usr/local/lib/python3.6/dist-packages/ansible
  executable location = /usr/local/bin/ansible
  python version = 3.6.9 (default, Nov  7 2019, 10:44:02) [GCC 8.3.0]

Yay.

Download your Modules

Let's check Ansible related Python modules next. For instance, I use the GCP Ansible modules quite a bit, so I need to install the Python v3 versions of the required modules.

# verify pip is also pointing to v3
pip --version
pip 9.0.1 from /usr/lib/python3/dist-packages (python 3.6)

# install Google Ansible required modules
sudo apt install python-requests
sudo pip install google-auth requests

Winning

That's about it. Just keep this change in mind as you do Python things and you won't have to worry about this until Python 4 cuts over in 10+ years. :-)
